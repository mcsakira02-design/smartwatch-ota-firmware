===============================================================================
                        EC S100 Watch Bootloader v1.0
                           初版引导加载程序说明文档
===============================================================================

项目信息:
---------
项目名称: 智能手表引导加载程序
版本号: v1.0
目标芯片: STM32F411xE
开发环境: Keil MDK-ARM

功能特性:
---------
1. 支持Ymodem协议固件下载
2. AES-256-CBC加密固件解密
3. 直接写入内部Flash App区域
4. 串口通信 (USART1, 115200bps)
5. 按键触发升级模式
6. 固件完整性验证
7. 自动跳转到应用程序

内存布局:
---------
Bootloader区域: 0x08000000 - 0x0800BFFF (48KB)
App应用区域:   0x0800C000 - 0x0807FFFF (464KB)
保留区域:      0x08018000 - 0x0807FFFF (已废弃)

硬件配置:
---------
串口: USART1 (PA9-TX, PA10-RX)
按键: 用户按键检测
LED:  状态指示
时钟: 外部25MHz晶振，系统96MHz

升级流程:
---------
1. 上电检测按键状态
2. 按键按下进入升级模式
3. 通过Ymodem协议接收加密固件
4. 实时AES解密并写入Flash
5. 验证固件完整性
6. 跳转到新应用程序

技术特点:
---------
- 取消双备份区设计，直接写入App区
- CBC模式AES解密，支持跨数据包处理
- 16字节对齐数据处理
- 从第一个数据包提取实际App大小
- 完整的错误处理和日志记录


注意事项:
---------
1. 固件必须使用AES-256-CBC加密
2. 加密密钥和IV已预设在代码中
3. 升级过程中请勿断电
4. 确保串口连接正常
5. 固件大小不得超过464KB
